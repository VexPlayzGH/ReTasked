|||   RETASKED ToDo App   |||

// This project is in an early state and may receive updates.

ReTasked is a Terminal UI App built using the Ratatui framework for Rust. The executable only works on Windows.
Once you run it, follow the instructions:

Press 'F1' or 'H' on your keyboard to view the list of available keybinds.
Happy note taking!
IMPORTANT: This App does not have auto-save, you will need to save it manually using 'S', it will then be written on the file 'output.txt'.
